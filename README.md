# **Havenlly - Airbnb Clone Project**

Explore the live version of Havenlly at [https://havenlly.onrender.com/](https://havenlly.onrender.com/).

**Havenlly** is a full-stack web application that replicates the core features of Airbnb, allowing users to explore, list, and book accommodations. This project is built using a variety of technologies and tools, including MongoDB, Node.js, Express.js, CSS, EJS, Mapbox, Cloudinary for cloud storage, and MongoDB Atlas for database hosting.

## Features

- **User Authentication**: Havenlly provides user registration and login functionality using Passport.js for authentication.

- **Accommodation Listings**: Users can create and manage accommodation listings, complete with descriptions, pricing, and images. Image uploads are handled using Multer and Multer Storage Cloudinary.

- **Interactive Maps**: The application utilizes Mapbox to display interactive maps, making it easy for users to visualize the location of accommodations.

- **Cloud Storage**: Images for accommodations are stored using Cloudinary, ensuring efficient and reliable cloud-based image hosting.

- **Session Management**: Express-session is used for session management, while Connect-Mongo provides a secure way to store session data in MongoDB.

- **Data Validation**: Joi is employed for data validation, enhancing the security and reliability of the application.

## Deployment

The project can be deployed using a service like [Render](https://render.com/) or any other hosting platform of your choice. Be sure to set up environment variables in your deployment environment as well.
